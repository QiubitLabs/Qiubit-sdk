# Qiubit React Hooks (@octra/react)

Simple Wagmi-style hooks for React integration.

## Installation
```bash
npm install QiubitLabs/Qiubit-sdk#main:packages/octra-react
```

## Usage
```jsx
import { OctraProvider, ConnectButton } from 'octra-react';

function App() {
  return (
    <OctraProvider>
      <ConnectButton />
    </OctraProvider>
  );
}
```
